# ChronoDots

**ChronoDots** is a premium Android calendar widget app featuring a "Perfect Dots" aesthetic.

## Features
- **Year View Widget**: 365-day visualization with footer text.
- **Deep Indigo Theme**: Premium dark mode design.
- **Customizable**: Emoji rules, layouts, and styles.

## Setup
1. Open in Android Studio.
2. Build and Run on a device/emulator (Android 13+ recommended).
